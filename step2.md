# Step 2: Creating Jekyll Site

Create a new site:
```bash
jekyll new demo_site
cd demo_site
```
---

Serve locaclly:
```bash
bundle exec jekyll serve
```
---

Visit: [http://localhost:4000](http://localhost:4000)
